# aiqicha_pac


#最近刚学了两周golang 感觉自己又行了 就花费几天时间边搜索库使用边写 这个小玩意勉强可以玩一下.

这个小程序可以通过搜索公司全称 获取企业架构，公司名称，网站地址，电话，邮箱。 备案信息，投资，控股，分支。

## 使用步骤
在aqc 登陆自己的账号 然后复制header头信息到cookie.json 文件里面 代码会自动读取设置请求头信息
